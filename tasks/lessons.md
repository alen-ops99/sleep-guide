# Lessons Learned — Croatian Sleep Guide

## Project-Specific Notes
- Somnia app (`/Sleep-Wellness/sleep-clinic-app/src/data/`) contains comprehensive clinical data that serves as fact-checked foundation
- Target audience: Croatian family medicine physicians with near-zero sleep medicine training
- Tone: plain-language, action-oriented, not textbook-academic
- Must check HALMED for Croatian drug availability (not all US/EU drugs are available)
- Croatian medical terminology: use INN drug names, standard Croatian diacritics (č, ć, š, ž, đ)
